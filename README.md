# BlueBirdsSongSystemReportUI
BlueBirdsSongSystemReportUI系统定时报告UI及混合生成模块,
基于SSL邮件系统，定时生成PDF报告,可以根据html或者xml生成PDF文件
基于BlueBirdsSong UI 开发,
BlueBirdsSong UI 灵感来自 Azone Blue Birds Song 美宇
